Output Signal Generation

Once all information is computed from the input signals and the optimum TDOA values are selected, it is time to output the enhanced signal and any accompanying information to be used by the subsequent systems. In this module several algorithms were used to account for the differences between the standard linear microphone array theory and the implementation in this module.
